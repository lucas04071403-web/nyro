# Acknowledgements

Nyro includes and depends on open-source work from many projects and
contributors.

Primary upstream projects:

- Hiddify App, the application this distribution is based on.
- Hiddify Core, the native proxy core integrated by this application.
- Sing-box, the universal proxy toolchain used by Hiddify Core.

Nyro also uses Flutter, Dart, Riverpod, gRPC, protobuf, Drift, Sentry SDK,
Flutter Distributor, and other open-source libraries listed in `pubspec.yaml`,
platform package manifests, and generated lock files.

Thank you to the original Hiddify, Hiddify Core, Sing-box, Flutter, and broader
open-source communities.
