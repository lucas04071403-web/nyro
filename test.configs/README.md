# Add Warp Config

## Step 1

Install Nyro from the GitHub releases page:

https://github.com/lucas04071403-web/nyro/releases/latest

Nyro is based on Hiddify App and powered by Hiddify Core and Sing-box. See
`NOTICE.md` for attribution details.

## Step 2

Copy the following URL and import it in Nyro:

```text
https://raw.githubusercontent.com/lucas04071403-web/nyro/main/test.configs/warp
```
